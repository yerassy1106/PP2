# Interactive Music Player

A keyboard-controlled music player that supports MP3 and WAV formats.

## Controls
- **P**: Play track
- **S**: Stop playback
- **SPACE**: Pause / Resume
- **N**: Next track
- **B**: Previous track (Back)

## Setup
Place your audio files in the `music/` folder inside this directory before launching.

## How to Run
```bash
python music_player/main.py
