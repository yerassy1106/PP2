# Practice 09: Game Development with Pygame - Part 1

This repository contains three Python applications developed using the Pygame library as part of the University Practice.

## Structure
- `mickeys_clock/`: A digital-style clock with Mickey Mouse hands.
- `music_player/`: An interactive music player with keyboard controls.
- `moving_ball/`: A simple game featuring boundary-checking logic.
