# Moving Ball Game

A simple interactive game focused on event handling and coordinate constraints.

## Task Requirements
- A red ball (radius 25) on a white background.
- Movement via Arrow Keys (Up, Down, Left, Right).
- Each step is 20 pixels.
- **Boundary Checking**: The ball cannot move outside the window frame.

## How to Run
```bash
python moving_ball/main.py
