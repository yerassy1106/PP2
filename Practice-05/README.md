# Practice 5 – Python Regular Expressions

## Objective
This practice focuses on using Regular Expressions in Python.

## Tasks

The program reads a receipt from raw.txt and extracts important data using the `re` module.

Extracted information:
- Product names
- Prices
- Total amount
- Payment method
- Date and time

## Files

receipt_parser.py – Python script that parses the receipt  
raw.txt – raw receipt text  
README.md – description of the project

## How to Run

```bash
python receipt_parser.py
